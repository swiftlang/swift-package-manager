# SwiftPM 6.4 Release Notes

Swift Package Manager add several new features in version 6.4.

The key features include:
- adding first class support for Software Bill of Materials (SBOM)
- making the Swift Build build system the default

## Software Bill of Materials (SBOM) support

Swift Package Manager can now generate SBOM for your Swift Package.  Read more [here](<doc:GeneratingSBOMs>).

## SwiftBuild: The new default build system

SwiftPM has changed it's default build system to Swift Build.  Read more [here](<doc:SwiftBuildDefault>).
